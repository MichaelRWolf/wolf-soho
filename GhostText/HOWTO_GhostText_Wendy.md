# HOWTO: Using GhostText (Wendy Edition)

This guide is for **everyday use**, not setup or troubleshooting.

GhostText is here for **one reason**:
> **So you never lose text you are typing into fields on web pages.**

You do not need to understand how it works. Just follow the steps.

---

## What GhostText Does (Plain Language)

- Web page text fields are **fragile**
- GhostText mirrors what you type into a **safe editor window**
- If the browser crashes, your words are **still there**

Think of it as:
> *A safety net for thinking and drafting on the web.*

## The Golden Rule

> **If the writing matters, use GhostText.**

Use GhostText when you are **actually thinking and drafting**, not just casually typing.

Why:
- While GhostText is active, the editor is holding your words in a **buffer**
- A *buffer* is safe, stable, and **not tied to a file yet**
- This is ideal for thinking, drafting, revising, and exploring language

If it‚Äôs quick, trivial, or disposable, you can type directly into fields on web pages.
